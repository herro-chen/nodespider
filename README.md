##NodeSpider是什么?

NodeSpider 是一个 nodejs 实现的爬虫系统

框架是express
核心模块是 superagent, cheerio, async

#### superagent
一个轻量的客户端请求代理模块

#### cheerio
服务端的jquery

#### async
流程控制模块
